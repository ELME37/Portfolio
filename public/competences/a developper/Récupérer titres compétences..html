    Il semble que vous souhaitiez récupérer le titre de vos compétences (skills) à partir d'un fichier JSON dans votre composant Skills. Vous avez déjà un code en place pour charger les données à partir du fichier JSON, mais il manque le code pour afficher les titres des compétences dans votre composant.

    Si les titres des compétences sont stockés dans les clés de votre objet JSON (par exemple, sous les clés "savoirFaire", "outils", etc.), vous pouvez les extraire et les afficher dans votre composant de la manière suivante :

    jsx
    // ... (importations et autres codes)

export default function Skills() {
    const [skillData, setSkillData] = useState([]);

    useEffect(() => {
        fetch('/skills.json')
            .then(response => response.json())
            .then(data => setSkillData(data))
            .catch(error => console.error(error))
    }, []);

    return (
        <div className='skills'>
            <Layout>
                {/* ... (autres éléments) */}
                <div className='skills__info'>
                    {/* Afficher les titres des compétences */}
                    <h2>Compétences</h2>
                    <ul>
                        {Object.keys(skillData).map(skillType => (
                            <li key={skillType}>{skillType}</li>
                        ))}
                    </ul>
                </div>
                {/* ... (autres éléments) */}
            </Layout>
            <Footer />
        </div>
    );
}

    Dans cet exemple, nous utilisons la méthode Object.keys(skillData) pour obtenir un tableau des clés de l'objet JSON skillData. Ensuite, nous utilisons map pour itérer à travers ce tableau et afficher les titres des compétences. Vous pouvez adapter cette logique en fonction de la structure spécifique de votre fichier JSON et de la manière dont les titres des compétences sont organisés.

    Pour afficher le titre de la compétence lorsque vous survolez une compétence spécifique, vous pouvez utiliser l'état local pour stocker le titre de la compétence survolée et le mettre à jour lorsque survolé. Voici comment vous pourriez le faire :

    jsx
    // ... (importations et autres codes)

export default function Skills() {
    const [skillData, setSkillData] = useState({});
    const [hoveredSkillTitle, setHoveredSkillTitle] = useState(null);

    useEffect(() => {
        fetch('/skills.json')
            .then(response => response.json())
            .then(data => setSkillData(data))
            .catch(error => console.error(error))
    }, []);

    const handleSkillHover = (title) => {
        setHoveredSkillTitle(title);
    };

    return (
        <div className='skills'>
            <Layout>
                {/* ... (autres éléments) */}
                <div className='skills__info'>
                    {/* Afficher le titre de la compétence survolée */}
                    {hoveredSkillTitle && <p>{hoveredSkillTitle}</p>}
                </div>
                {/* ... (autres éléments) */}
                <div className='wheels'>
                    {/* ... (autres éléments) */}
                    {Object.keys(skillData).map(skillType => (
                        <div className='wheels__part1' key={skillType}>
                            {skillData[skillType].map(skill => (
                                <Wheel
                                    key={skill.id}
                                    title={skillType}
                                    skills={[skill]}
                                    styles={skillType}
                                    onMouseEnter={() => handleSkillHover(skill.title)}
                                    onMouseLeave={() => handleSkillHover(null)}
                                />
                            ))}
                        </div>
                    ))}
                    {/* ... (autres éléments) */}
                </div>
            </Layout>
            <Footer />
        </div>
    );
}

    Dans cet exemple, nous avons ajouté un état hoveredSkillTitle pour stocker le titre de la compétence survolée. Lorsque vous survolez une compétence, la fonction handleSkillHover est appelée, mettant à jour l'état avec le titre de la compétence survolée. Lorsque vous ne survolez plus une compétence, l'état est réinitialisé à null.

    Nous avons également ajouté des événements onMouseEnter et onMouseLeave aux éléments <Wheel> (j'assume que c'est le composant que vous utilisez pour afficher les compétences) pour mettre à jour l'état lorsque vous survolez ou quittez une compétence.

    Notez que vous devrez peut-être ajuster le code en fonction de la manière dont vous structurez vos composants et la manière dont le composant <Wheel> gère les événements.
